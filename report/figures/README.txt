Place exported diagram images here before compiling.

Required filenames (PNG or PDF recommended):
  class_diagram_internal.png   <- from diagrams/01_class_diagram_internal.md
  class_diagram_public.png     <- from diagrams/02_class_diagram_public.md
  state_diagram.png            <- from diagrams/03_state_diagram.md
  seq_success.png              <- from diagrams/04_sequence_success.md
  seq_retry_success.png        <- from diagrams/05_sequence_retry_success.md
  seq_circuit_open.png         <- from diagrams/06_sequence_circuit_open.md
  deployment_diagram.png       <- from diagrams/07_deployment_diagram.md

How to export from Mermaid:
  1. Open https://mermaid.live
  2. Paste the mermaid code block from the .md file
  3. Click "Export" -> PNG
  4. Save with the filename above into this folder
